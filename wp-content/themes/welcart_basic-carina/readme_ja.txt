******************************************
　　Welcart Carina Theme
******************************************

このデザインテーマは、Welcart 専用のネットショップテンプレートです。
Welcart Basic（Free） を親テーマとして作成された子テーマで、親テーマ（Welcart Basic）に対して
様々な拡張を行っています。詳しくはドキュメントをご覧ください。

御購入後にアップデータが有った場合は、マイページより最新版をダウンロードすることができます。
親テーマ Welcart Basic（Free）が必要です。
http://www.welcart.com/archives/4488.html

== インストール ==
1.管理パネルの[ 外観 ]＞[ テーマ ]ページ上部にある「新規追加」をクリックしてください。
2.テーマを追加ページ上部にある「テーマのアップロード」をクリックしてテーマの圧縮ファイルをそのままアップロードします。
3.アップロード完了後に該当テーマを「有効化」します。

同じテーマをアップロードする場合は、既存のテーマを一旦削除してください。
テーマを削除するとテーマに施した修正内容が全て消えてしまうのでご注意ください。
カスタマイズしたテーマはバックアップを取ってから削除・上書きを行ってください。



== ご利用上の注意点 ==
・WordPress4.4以降・Welcart1.7以降の環境でご利用ください。
・事前に Welcart Basic（Free）をインストールしておく必要があります。
・対応ブラウザはChrome、FireFox、Safari、Opera、IE9以上です
